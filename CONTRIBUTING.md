# 贡献指南 / Contributing Guide

## 中文

感谢你对 CloudMate 的关注！我们欢迎任何形式的贡献。

### 如何贡献？

1. **报告 Bug**
   - 在 GitHub Issues 中创建新 Issue
   - 描述问题的清晰步骤、预期结果和实际结果

2. **建议新功能**
   - 在 Issues 中提出你的建议
   - 说明为什么这个功能有用，如何使用

3. **提交代码**
   - Fork 项目
   - 创建新分支 (`git checkout -b feature/AmazingFeature`)
   - 提交更改 (`git commit -m 'Add AmazingFeature'`)
   - 推送到分支 (`git push origin feature/AmazingFeature`)
   - 开启 Pull Request

### 代码规范

- 使用 4 个空格缩进
- 添加必要的注释
- 保持代码风格一致
- 测试更改后再提交

---

## English

Thank you for being interested in CloudMate! We welcome all kinds of contributions.

### How to Contribute?

1. **Report Bugs**
   - Create a new Issue on GitHub
   - Describe the steps, expected result, and actual result clearly

2. **Suggest New Features**
   - Propose your ideas in Issues
   - Explain why this feature is useful and how to use it

3. **Submit Code**
   - Fork the project
   - Create a new branch (`git checkout -b feature/AmazingFeature`)
   - Commit your changes (`git commit -m 'Add AmazingFeature'`)
   - Push to the branch (`git push origin feature/AmazingFeature`)
   - Open a Pull Request

### Code Style

- Use 4 spaces for indentation
- Add necessary comments
- Keep code consistent
- Test your changes before submitting
